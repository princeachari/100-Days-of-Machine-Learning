# 📚 Day 005 - Student Productivity Prediction

## 📌 Project Overview

This project predicts student productivity using machine learning regression techniques. It demonstrates a complete end-to-end machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, hyperparameter tuning, cross-validation, and model evaluation.

## 📂 Dataset

- **Dataset:** Student Productivity Dataset
- **Target Variable:** Productivity

### Features include:
- Study Hours
- Sleep Hours
- Attendance
- Previous Scores
- Extracurricular Activities
- Internet Usage
- Mental Health Rating
- Physical Activity
- And other student-related attributes

- XGBoost

## 📊 Models Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- XGBoost Regressor

## 📈 Evaluation Metrics

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score
- Cross Validation Score

## 🏆 Results

XGBoost Regressor achieved the best overall performance after hyperparameter tuning, outperforming the other regression models on the Student Productivity dataset.
