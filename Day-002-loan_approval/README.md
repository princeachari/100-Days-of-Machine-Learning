# Day 02 - Loan Approval Prediction (Data Preprocessing & EDA)

## 📌 Overview

On Day 02, I worked on building the foundation of a Loan Approval Prediction project. The focus was on understanding the dataset, cleaning the data, handling missing values, encoding categorical features, performing exploratory data analysis (EDA), and preparing the data for machine learning models.

---

## 📚 Topics Covered

- Importing required libraries
- Loading the dataset
- Exploring the dataset
- Data cleaning
- Handling missing values
- Exploratory Data Analysis (EDA)
- Label Encoding
- One-Hot Encoding
- Feature Scaling using StandardScaler
- Train-Test Split

---

- Loan Approval distribution
