# 🍎 Day 004 - Apple Product Price Prediction

## 📌 Project Overview

This project predicts the selling price of Apple products using machine learning regression models. It follows a complete regression workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and model evaluation.

## 📂 Dataset

- Source: Apple Products Price Dataset (Kaggle)
- Features include:
  - Platform
  - Product Category
  - Model Name
  - Condition
  - Launch Price (USD)
  - Rating
  - Reviews Count
  - Sale Event
  - Stock Status
  - Date

- **Target Variable:**
  - Selling Price (INR)

## 🛠 Technologies Used

- Python

- Pandas
- Matplotlib
- Seaborn

- XGBoost

## 🤖 Machine Learning Workflow

- Data Loading
- Data Exploration
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Missing Value Handling
- Feature Engineering
- Feature Encoding
- Train-Test Split
- Model Training
- Model Comparison
- Model Evaluation

## 📊 Models Used

- Linear Regression
- Random Forest Regressor
- XGBoost Regressor

## 📈 Evaluation Metrics

- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## 🏆 Results

The performance of Linear Regression, Random Forest Regressor, and XGBoost Regressor was compared using regression evaluation metrics. The best-performing model was selected based on R² Score, MAE, and RMSE.
