# 📅 Day 3 - Walmart Sales Prediction

## 📌 Project Overview

On Day 3, I built a machine learning regression model to predict Walmart weekly sales. The project covers the complete ML pipeline, from data preprocessing and exploratory data analysis (EDA) to model training, hyperparameter tuning, and evaluation.

---

## 🎯 Objective

Develop a regression model that accurately predicts Walmart weekly sales using historical store and economic data.

---

## 📂 Dataset Features

- Store
- Holiday Flag
- Temperature
- Fuel Price
- CPI
- Unemployment
- Weekly Sales (Target)

---

## 🛠️ Skills Practiced

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Feature Scaling
- Model Training
- Hyperparameter Tuning
- Model Evaluation

---

## 🤖 Models Used

- Linear Regression
- Ridge Regression
- Support Vector Regression (SVR)

---

## 📊 Results

| Model | R² Score |
|--------|---------:|
| Ridge Regression | **0.924** |
| Support Vector Regression | **0.696** |

🏆 **Best Performing Model:** Ridge Regression

---

## 📚 Key Learnings

- Importance of data preprocessing before model training.
- Comparison of multiple regression algorithms.
- Hyperparameter tuning using **GridSearchCV**.
- Understanding model evaluation using the **R² Score**.
- Selecting the best model based on generalization performance.

---

## 📌 Conclusion

This project demonstrates an end-to-end machine learning workflow for a regression problem. After comparing multiple models, **Ridge Regression** achieved the highest performance with an **R² Score of 0.924**, making it the most reliable model for predicting Walmart sales. The project reinforced the importance of preprocessing, feature engineering, and hyperparameter tuning in building accurate and robust machine learning models.
